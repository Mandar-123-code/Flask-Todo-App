About This Todo Application

This is a simple yet functional TODO list web application built using the Python Flask web framework. It allows users to create, update, delete, and search their daily tasks in an organized manner.

The app is designed to help users manage their personal or work-related tasks efficiently. Whether you need to keep track of grocery items, daily goals, assignments, or project tasks — this application helps you stay productive and on schedule.

🔧 Features:
  • 📝 Add a new task with a title and description
  • 🗑️ Delete tasks that are completed
  • ✏️ Update existing tasks with new details
  • 🔍 Search functionality to quickly find your todos
  • 📅 Timestamps to track when tasks were created
🛠️ Technologies Used:
  • Frontend: HTML, CSS, Bootstrap 5
  • Backend: Python with Flask
  • Database: SQLite with SQLAlchemy ORM

This project is ideal for beginners learning Flask, web development, or backend frameworks. It’s a great example of how to connect forms, databases, and routing in a real-world project.
